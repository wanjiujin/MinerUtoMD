# MinerUtoMD 开发与打包指南

## 一、项目概述

**项目名称：** MinerUtoMD - 智能文档转换工具

**核心功能：**
- PDF 转 Word/Markdown/HTML/EPUB
- 公式识别（LaTeX）
- 表格识别与转换
- 图片提取与嵌入
- OCR 支持（中文）

**技术栈：**
- Python 3.10/3.11
- MinerU 3.1.15（PDF 提取）
- Pandoc 3.9.0.2（格式转换）
- PyTorch 2.12.0
- tkinter（GUI）

### 3.2 MinerU 提取器（mineru_extractor.py）

**关键功能：**
- 自动检测 MinerU 版本（3.x / 1.x）
- 设置环境变量
- 构建 CLI 命令

```python
class MinerUExtractor:
    # 新版 MinerU 3.x CLI 路径
    MINERU_CLI_V3 = r"C:\ProgramData\miniforge3\envs\mineru2\Scripts\mineru.exe"
    
    def _setup_environment(self) -> dict:
        """设置环境变量"""
        env = os.environ.copy()
        env['HF_ENDPOINT'] = 'https://hf-mirror.com'
        env['HF_HOME'] = 'C:/hf_cache'
        return env
    
    def _build_v3_command(self, pdf_path, output_dir, **kwargs):
        """构建 MinerU 3.x 命令"""
        cmd = [
            self.cli_path,
            '-p', str(pdf_path),
            '-o', str(output_dir),
            '-b', 'pipeline',      # 本地模型后端
            '-f', 'true',          # 公式识别
            '-t', 'true',          # 表格识别
            '-l', 'ch'             # 中文 OCR
        ]
        return cmd
```

### 3.3 Pandoc 转换器（pandoc_converter.py）

**关键功能：**
- 图片嵌入到 Word
- 设置工作目录（解决相对路径问题）

```python
def markdown_to_word(self, md_path, output_path, images_dir=None):
    """Markdown 转 Word，嵌入图片"""
    extra_args = []
    
    # 构建资源路径
    resource_paths = [str(md_path.parent)]
    if images_dir:
        resource_paths.append(str(images_dir))
    
    extra_args.extend(['--resource-path', ':'.join(resource_paths)])
    
    # 设置工作目录为 Markdown 文件所在目录
    cwd = str(md_path.parent)
    
    return self.convert(
        md_path, output_path,
        input_format='markdown',
        output_format='docx',
        cwd=cwd,
        extra_args=extra_args
    )
```

### 3.4 Markdown 优化器（markdown_optimizer.py）

**关键功能：**
- HTML 表格转 Markdown 表格
- 表格前后添加空行

```python
def _convert_html_tables(self, content: str) -> str:
    """将 HTML 表格转换为 Markdown 表格"""
    def replace_table(match):
        table_html = match.group(0)
        md_table = self._html_table_to_markdown(table_html)
        # 确保表格前后有空行
        if md_table:
            return '\n\n' + md_table + '\n\n'
        return ''
    
    content = re.sub(r'<table>.*?</table>', replace_table, 
                     content, flags=re.DOTALL | re.IGNORECASE)
    return content
```

### 3.5 工作流（workflow.py）

**关键功能：**
- 协调各模块
- 复制图片文件夹
- 处理输出

```python
def run(self, pdf_path, output_dir, output_formats, **kwargs):
    # 步骤1: PDF → Markdown
    extract_result = self.mineru.extract(pdf_path, temp_dir, **kwargs)
    
    # 步骤2: 优化 Markdown（在原目录进行）
    optimized_md_path = md_path.parent / f"{pdf_name}_optimized.md"
    self.optimizer.optimize(md_path, optimized_md_path)
    
    # 步骤3: 转换为目标格式
    for fmt in output_formats:
        if fmt == 'md':
            # 复制 Markdown 和图片文件夹
            shutil.copy2(optimized_md_path, output_dir)
            shutil.copytree(images_dir, output_dir / 'images')
        elif fmt == 'docx':
            self.pandoc.markdown_to_word(
                optimized_md_path, output_path, 
                images_dir=images_dir
            )
```

## 五、打包 EXE

### 5.1 安装 PyInstaller

```bash
pip install pyinstaller
```

### 5.2 打包命令

```bash
pyinstaller --onefile --windowed \
    --name "MinerUtoMD" \
    --add-data "config.yaml;." \
    --add-data "workflow.py;." \
    --add-data "mineru_extractor.py;." \
    --add-data "pandoc_converter.py;." \
    --add-data "markdown_optimizer.py;." \
    gui.py --clean
```

### 5.3 打包参数说明

| 参数 | 说明 |
|------|------|
| `--onefile` | 打包成单个 exe 文件 |
| `--windowed` | 不显示控制台窗口 |
| `--name` | 输出文件名 |
| `--add-data` | 添加数据文件（格式：`源路径;目标路径`） |
| `--clean` | 清理临时文件 |

### 5.4 常见打包问题

**问题 1：exe 运行时找不到模块**

解决方案：使用 `--add-data` 添加所有依赖的 Python 文件

**问题 2：exe 文件过大**

解决方案：使用 `--exclude-module` 排除不需要的模块

```bash
pyinstaller --onefile --windowed \
    --exclude-module matplotlib \
    --exclude-module PIL \
    ...
```

**问题 3：杀毒软件误报**

解决方案：使用代码签名证书签名 exe

## 七、关键问题解决方案

### 7.1 HuggingFace 无法访问

**问题：** 国内无法直接访问 HuggingFace

**解决方案：**

```python

os.environ['HF_ENDPOINT'] = 'https://hf-mirror.com'
```

### 7.2 Windows 路径长度限制

**问题：** Windows 默认路径长度限制 260 字符

**解决方案：**

```python
import os
os.environ['HF_HOME'] = 'C:/hf_cache'  # 使用短路径
```

### 7.3 图片未嵌入 Word

**问题：** Word 中图片显示为占位符

**解决方案：**

1. 设置 Pandoc 工作目录为 Markdown 文件所在目录
2. 添加资源搜索路径

```python
# 设置工作目录
cwd = str(md_path.parent)

# 添加资源路径
extra_args.extend(['--resource-path', str(md_path.parent)])
extra_args.extend(['--resource-path', str(images_dir)])
```

### 7.4 表格显示为纯文本

**问题：** HTML 表格未转换为 Markdown 表格

**解决方案：**

1. 将 HTML `<table>` 转换为 Markdown 表格
2. 表格前后添加空行

```python
# 转换后添加空行
return '\n\n' + md_table + '\n\n'
```

### 7.5 Markdown 中图片不显示

**问题：** Markdown 文件中图片路径正确但不显示

**解决方案：**

复制图片文件夹到输出目录：

```python
if fmt == 'md':
    shutil.copy2(optimized_md_path, output_dir)
    # 复制图片文件夹
    shutil.copytree(images_dir, output_dir / 'images')
```
